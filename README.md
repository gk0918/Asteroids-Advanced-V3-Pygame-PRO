# Asteroids-Advanced-V3-Pygame-PRO

ASTEROID ADVANCED V3 – PYGAME PRO:

Asteroid Advanced V3 Pygame PRO is an advanced 2D space arcade game built using Python and Pygame.
The player controls a spaceship in open space and must destroy incoming asteroids while surviving as long as possible. This project demonstrates professional game structure, smooth controls, collision physics, sound effects, scoring, levels, and advanced gameplay mechanics.
It is designed as a portfolio-quality project to showcase strong Python, Pygame, and game development skills.

GAME OVERVIEW:

Control a spaceship in space.
Asteroids spawn from different directions.
Shoot and destroy asteroids before they hit your ship.
Game difficulty increases over time.
Score increases for every asteroid destroyed.
Survive as long as possible to achieve a high score.

KEY FEATURES:

Smooth spaceship movement with rotation.
Realistic asteroid movement and collision physics.
Bullet shooting system.
Particle effects / explosion effects.
Sound effects and background music.
Scoreboard and high-score tracking.
Increasing difficulty and speed levels.
Game Over and restart system.
Clean multi-file project structure (OOP based).

TECHNOLOGIES USED:

Python 3.
Pygame.
Object-Oriented Programming (OOP).

HOW TO RUN THE GAME:

Step 1: Install Python
Ensure Python 3 is installed.

Step 2: Install Pygame
Open terminal / command prompt:
pip install pygame

Step 3: Run the Game
python main.py

CONTROLS:

Key                   Action
↑ Arrow            Move forward.
← / → Arrows       Rotate spaceship.
Spacebar           Shoot bullets.
R                  Restart game.
Q                  Quit game.
